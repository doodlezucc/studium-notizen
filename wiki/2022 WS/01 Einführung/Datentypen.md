# Datentypen

## Java
### Primitive Datentypen
- **Numerisch**
	- Ganzzahlen: `byte, short, int, long, char`
	- Gleitkomma: `float, double`
- **Andere**
	- `boolean`

### Referenz-Datentypen
- Arrays
- Klassen/Objekte
- Interfaces
- `null`-Datentyp

## Python
Primitive Typen entsprechen **immutable types**.

| Typ       | Immutable |
| --------- | --------- |
| bool      | x         |
| int       | x         |
| float     | x         |
| **list**    |           |
| tuple     | x         |
| str       | x         |
| **set**     |           |
| frozenset | x         |
| **dict**    |           |
| **Klassen/Objekte**    |           |
